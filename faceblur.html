About the Project

This project automatically detects and blurs faces from webcams using OpenCV and MediaPipe libraries in Python. It is designed to protect privacy during video calls or recordings by applying Gaussian blur on detected facial areas.

The program uses advanced facial landmark detection to create custom regions for accurate blurring, making it suitable for both individual and multiple faces in real-time video streams.

Features of the Project

  • Real-time face detection and blurring.
  • Supports multiple faces simultaneously.
  • Accurate facial region detection using MediaPipe.
  • Adjustable blur intensity for customization.
