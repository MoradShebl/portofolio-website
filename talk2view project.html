About the Project

Talk2View is an innovative app designed to bridge communication gaps by converting spoken words into animated sign language and text. It leverages advanced speech recognition technology and a user-friendly interface to enhance accessibility for individuals with hearing impairments. The app's design focuses on clarity, responsiveness, and seamless functionality, offering an inclusive solution for fostering better communication.

Talk2View highlights my dedication to innovation and accessibility, serving as a groundbreaking platform for enhancing communication. It offers features such as speech-to-sign conversion, text display, and a responsive design to create an inclusive experience for users.

Features of Talk2View

  • Speech-to-sign-language conversion for enhanced accessibility.
  • Real-time speech-to-text transcription for clear communication.
  • User-friendly and interactive interface for effortless navigation.
  • Built with Kivy for a modern and responsive design across devices.
